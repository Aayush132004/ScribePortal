The Scribe Booking Portal is a full-stack MERN application designed to assist specially-abled students in booking verified scribes for examinations. The platform follows WCAG 2.0 accessibility guidelines to ensure compatibility with assistive technologies, including screen readers, keyboard navigation, and high-contrast interfaces.
The project aligns with inclusive digital initiatives supported by the Government of India for physically- and visually-disabled learners.

Key Features

WCAG 2.0 accessibility compliance (semantic HTML, ARIA attributes, proper color contrast, focus management, keyboard operability)

Student and scribe authentication (JWT-based)

Real-time chat and audio/video calling (WebRTC, Socket.io)

Scribe rating and review system

Location-based scribe matching

Exam and booking management
Tech Stack

Frontend: React, Tailwind/SCSS, Context API/Redux
Backend: Node.js, Express.js
Database: MongoDB
Real-Time Services: WebRTC, Socket.io

Purpose

To provide an accessible, reliable, and inclusive platform for specially-abled students to find and coordinate with scribes efficiently, ensuring equal opportunities in academic examinations.
